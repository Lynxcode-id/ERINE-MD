## 🌙 Tentang Bot

**ERINE-MD PROJECT** adalah bot WhatsApp multi-device dengan sistem plugin yang fleksibel.  
Bot ini dibuat untuk hiburan, utilitas, downloader, dan otomatisasi grup maupun chat pribadi.

Cocok digunakan untuk:
- 📥 Download media
- 🎮 Game & fun command
- 👥 Manajemen grup
- ✨ AI tools
- 🖼️ Sticker maker

## ✨ Fitur Utama

Bot ini dilengkapi dengan berbagai fitur yang terorganisir dalam plugin.

| Kategori           | Deskripsi                                                      |
| ------------------ | -------------------------------------------------------------- |
| **AI Tools**        | Chat AI, prompt tools, generator teks.                        |
| **Downloader**      | TikTok, YouTube, Instagram, Facebook, dll.                    |
| **Hiburan**         | Truth, Dare, Tebak Gambar, RPG, dan game lainnya.             |
| **Manajemen Grup**  | Welcome, Anti Link, Kick, Hidetag, dll.                       |
| **Utilitas**        | QR Generator, Convert Tools, Media Tools.                     |
| **Sticker**         | Sticker maker + watermark.                                   |
| **Sistem Level**    | XP, level, limit, profile user.                               |
| **Owner Tools**     | Broadcast, reset data, maintenance.                           |

## 💡 Contoh Penggunaan

### ▶️ Membuat Sticker
Kirim gambar dengan caption:
```
.sticker
.brat
.s
.bratvid
```

### ▶️ Download Video TikTok
```
.tiktok https://vt.tiktok.com/xxxx
.tthd
.tt
```

### ▶️ Download Video Instagram
```
.ig https://www.instagram.com/p/xxxx
.ig story 
```

### ▶️ Download Lagu
```
.play judul lagu
.play3 <- versi full button 
```

### ▶️ Chat AI
```
.ai rekomendasi anime romance
.gpt
.gemini
```

### ▶️ Game & Fun
```
.truth
.dare
.tebakgambar
```

### ▶️ Menampilkan Menu
```
.menu
.menu all
.menu tags
```
